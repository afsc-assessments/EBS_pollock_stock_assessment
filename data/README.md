# Shareable data

Store only small, redistributable data objects required to demonstrate or test
the workflow. Do not commit confidential fishery, observer, or credentialed
database extracts.

For external data, commit a manifest entry containing the immutable object name,
owner, access method, version or extraction date, checksum when permitted, and
the script that produces the model-ready input.

This example contains manifests only. The actual 2024 model-ready files remain in
the authoritative assessment repository.
