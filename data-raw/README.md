# Input construction and provenance

This directory should contain shareable scripts—not confidential raw records—that
retrieve, clean, aggregate, and validate the annual inputs listed in
`../assessment-years/2024/data-manifest.csv`.

Each process should record:

- provider and responsible analyst;
- query or delivery identifier;
- extraction and freeze dates;
- temporal, spatial, species, and sampling filters;
- units and transformations;
- source and output checksums;
- confidentiality and redistribution restrictions.

The public 2024 repository includes model-ready products but does not provide a
complete, uniform provenance chain for every upstream delivery. Those missing
connections are made explicit in the manifest rather than inferred here.
